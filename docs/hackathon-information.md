### 「体験」にはどんなものがあるか

| カテゴリ | 具体例 |
| :--- | :--- |
| **交通** | 駅、電車、空港、飛行機、車 |
| **宿泊** | ホテル、旅館 |
| **飲食** | レストラン、カフェ、コンビニ |
| **案内・情報** | マップ、案内所、マナー |
| **店舗** | お店・空港免税・地場産品・配送、現金、両替、キャッシュレス |
| **コミュニケーション** | 外国、視覚・聴覚、文化 |

### 作るものを決めるヒント

1. **登場人物はだれ？誰が使う？**
   だれが使う？その人の周りにはどんな人がいる？

2. **なんのために使う？**
   「あのとき○○だったら良かったのに」「もっとこうだったら最高なのに」
   我慢したこと、諦めたこと／諦めさせたことはなに？

3. **何を作る？**
   - 情報が届けば解決する？
   - 人が繋がれば解決する？
   - 見える化すれば解決する？
   - 2日間で「動くもの」にするなら、一番大事な機能は何？

### 参考事例

- **ケネディ宇宙センター**: 未来の宇宙探査をVR体験できる「HYPERDECK VR」
- **韓国・ソウルの「スマート観光地」**: Wi-Fiやビッグデータ、IoTを活用して観光客にパーソナライズされた体験を提供

> [!TIP]
> どんなものがあるのか、普段使っているものを「迎える側」の視点でよく観察して、考えてみましょう！
